# 🦅 Aegis Eagle

An AI-powered 3D-printed Eagle Security Robot designed for surveillance, motion detection, and smart monitoring.

## Features
- 🎥 Live video streaming
- 🧠 AI human detection
- 👁️ Motion sensing
- 🔊 Voice alerts
- 💡 LED eagle eyes
- 📱 Mobile notifications
- 🖨️ Fully 3D printable design

## Hardware
- Raspberry Pi 5
- ESP32
- Camera Module
- PIR Sensor
- 18650 Batteries
- Servo Motors

## Software
- Python
- OpenCV
- TensorFlow Lite
- Flask
